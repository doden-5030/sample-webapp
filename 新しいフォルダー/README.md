# Game Strategy Guide

シンプルなHTMLベースのゲーム攻略Tipsアプリです。

## 機能

- 🎮 ゲームタイトル検索
- 🔍 キーワード検索で絞り込み
- 📱 レスポンシブ対応（スマホ/タブレット/PC）
- 📋 モーダルでTips詳細表示
- ✨ ビルドツール不要（HTMLだけで動作）

## 使い方

1. `index.html` をブラウザで開く
2. ゲームタイトルやキーワードで検索
3. Tipsカードをクリックして詳細を表示

## ファイル構成

```
├── index.html       # HTMLマークアップ
├── style.css        # スタイル・レスポンシブデザイン
├── app.js           # ロジック・イベントハンドリング
└── README.md        # このファイル
```

## 特徴

- 外部ライブラリ不要
- ローカルファイルで完全動作
- サンプルデータ付属（app.js内）
- アクセシビリティ対応（aria属性）
- XSS対策実装

## カスタマイズ

`app.js` の `TIPS` 配列にゲーム情報を追加することでコンテンツを拡張できます。

```javascript
{
  id: 6,
  game: "ゲームタイトル",
  title: "Tips のタイトル",
  summary: "短い説明",
  content: "詳細な内容",
  tags: ["タグ1", "タグ2"],
  date: "YYYY-MM-DD"
}
```
